JavaScript Solution: Balancing Strings with '<' and '>'

About the Algorithm

This JavaScript solution determines whether strings consisting of '<' and '>' characters can be balanced using a given number of replacements. The function checks balance, processes replacements, and outputs the results for various inputs.

How the Algorithm Works

  1. Iterates through each string in the array.
  2. Keeps track of unmatched '>' characters and calculates necessary replacements.
  3. Compares required replacements with the allowed maximum replacements.
  4. Outputs `1` for balanced strings and `0` for unbalanced strings.

Code Implementation

function balancedOrNot(expressions, maxReplacements) {
    return expressions.map((expression, index) => {
        let stack = 0; 
        let replacements = 0;

        for (const char of expression) {
            if (char === '<') {
                stack++;
            } else if (char === '>') {
                if (stack > 0) {
                    stack--;
                } else {
                    replacements++;
                }
            }
        }

        return stack === 0 && replacements <= maxReplacements[index] ? 1 : 0;
    });
}

// Example usage:
const expressions = ['<>>>', '<>>>>'];
const maxReplacements = [2, 2];
console.log(balancedOrNot(expressions, maxReplacements)); // Output: [1, 0]
